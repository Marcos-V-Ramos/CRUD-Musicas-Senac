Como iniciar o projeto:

Para iniciar o projeto, basta importar o projeto Spring Boot em alguma IDE de desenvolvimento Java (projeto testado apenas no IntelliJ Community), baixar as dependências, iniciar o projeto e acessar dinâmicamente o arquivo listagem.html (http://localhost:8080/listagem.html)para dar inicio ao fluxo do mesmo.

Observações

  • Tanto a parte de estilização (CSS) quanto a parte de programação (JavaScript) foram desenvolvidas dentro do seu respectivo arquivo HTML, visando a diminuição de arquivos finais;
  • A área de LocalStorage foi utilizada como estratégia para a lógica da caixa de sucesso ao adicionar as duas primeiras músicas.
